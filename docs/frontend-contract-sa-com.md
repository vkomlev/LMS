# Контракт для фронта: ответы SA_COM (задачи с комментарием)

**Эндпоинт:** `POST /api/v1/attempts/{attempt_id}/answers`

## Формат ответа для типа SA_COM

- **`response.value`** — обязательно. Основной ответ ученика (участвует в проверке и баллах).
- **`response.comment`** — опционально (`string | null`). Комментарий ученика: только хранится и возвращается в API, **в расчёте баллов не участвует**.

## Пример тела запроса

```json
{
  "items": [
    {
      "task_id": 2,
      "answer": {
        "type": "SA_COM",
        "response": {
          "value": "основной ответ",
          "comment": "комментарий ученика"
        }
      }
    }
  ]
}
```

Без комментария (обратная совместимость):

```json
{
  "type": "SA_COM",
  "response": {
    "value": "основной ответ"
  }
}
```

## Выдача

Во всех ответах API, где возвращается `answer_json` или результаты попытки (в т.ч. `GET /attempts/{id}`, `POST /attempts/{id}/finish`), комментарий приходит в том же месте: **`answer_json.response.comment`**.

## Кратко

| Поле            | Обязательность | Роль                          |
|-----------------|----------------|-------------------------------|
| `response.value`   | обязательно    | Проверка и баллы              |
| `response.comment`| опционально    | Хранение и выдача, не scoring |
