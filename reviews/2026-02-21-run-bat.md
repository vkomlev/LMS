# Добавлен run.bat для запуска с виртуальной средой

**Дата:** 2026-02-21

## Контекст

Добавлен batch-файл `run.bat` в корень проекта для запуска приложения с автоматической инициализацией виртуальной среды Python (venv): создание venv при отсутствии, активация, при наличии `requirements.txt` — установка зависимостей, запуск `python run.py`.

## Изменения

- Новый файл: `run.bat`
