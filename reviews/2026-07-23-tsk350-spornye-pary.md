# tsk-350 — спорные пары заданий ЕГЭ (не тронуты)

**69 пар**, где критерий НЕ уверен, дубль это или вариация. Ничего не скрыто — решение за оператором. У каждого задания: ссылка на страницу в LMS (под своей сессией) и ссылка на первоисточник для сверки.

> Это уже отфильтрованный список. Пары, где числа или формула условия явно разошлись (разные задания одного типа — например, «убрать 4/7 камня» против «убрать 3/5»), сюда НЕ попали: они уверенно не дубли. Здесь только пары, где по всем доступным факторам сходится, но одного подтверждения не хватает.

Первоисточники: **КомпЕГЭ** — API-страница с условием и ответом (`key`); **Решу ЕГЭ** и **Поляков** — обычные страницы задачи; **ТГ-пост** — разбор в @cyberguru_ege; **WP-навигатор** — авторская страница на сайте, ссылка ведёт на её первоисточник; **Крылов** — книжный сборник (веб-страницы нет, дана ссылка на видеоразбор, если он есть).

## Как быстро принять решение

Откройте оба задания в LMS рядом. Одинаковые условие + числа + вопрос → дубль (скажите — скрою слабейший по тому же критерию канона). Разные числа или разный вопрос → это вариации, оставляем оба.

Сгруппировано по причине сомнения, приоритетные — сверху.
